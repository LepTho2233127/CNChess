# CNChess
Projet de S4 

AVEC POWERSHELL

1. Cloner le projet
    git clone <URL_DU_REPO>
    cd CNChess

2. Install docker

3. Build docker image
```bash
sudo docker compose build
```

4. Run code
```bash
sudo docker compose run cnchess
```

5. Close docker (or ctrl-c)
```bash
sudo docker compose down
```
